Gallery

The EpiModel Gallery features short vignettes, tutorials, and other case studies illustrating how to use EpiModel to perform specific research tasks, including writing extension modules for particular disease types. We recently migrated the Gallery over to a Github repository for easier communication with users. Contributions and requests by users are welcome, following the instructions on the main repository page.
